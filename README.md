
# AI-Powered Resume Analyzer Using NLP

## Project Overview
This project analyzes resumes using NLP and matches them with job descriptions.

## Goal
To automate candidate screening using AI.

## Technologies Used
- Python
- NLP (NLTK / spaCy)
- Scikit-learn

## Features
- Resume parsing
- Text preprocessing
- TF-IDF feature extraction
- Similarity scoring
- Candidate ranking

## How to Run

1. Install required libraries:
   pip install nltk scikit-learn

2. Run the project:
   python project.py

## Example

**Resume:**
Python developer with NLP skills

**Job Description:**
Looking for Python NLP developer

**Output:**
Similarity Score: 85%  
Strong Match
